Αναγνώριση αντικειμένων σε κινητή συσκευή με χρήση Machine Learning

Μια έξυπνη, offline εφαρμογή κινητής τηλεφωνίας που χρησιμοποιεί τεχνητή νοημοσύνη στη συσκευή για να ανιχνεύσει ζημιές αυτοκινήτων σε πραγματικό χρόνο. Σχεδιάστηκε για ασφαλιστικούς εκτιμητές, εταιρείες ενοικίασης αυτοκινήτων και επιθεωρητές οχημάτων, αυτή η εφαρμογή παρέχει άμεσα αποτελέσματα ανίχνευσης ζημιών με βαθμολογία εμπιστοσύνης, παρακολούθηση FPS σε πραγματικό χρόνο και αυτόματη καταγραφή αποτελεσμάτων.

Επισκόπηση
Ο χειροκίνητος έλεγχος ζημιών αυτοκινήτων είναι αργός, υποκειμενικός και επιρρεπής σε ανθρώπινα λάθη. Αυτό το έργο αντικαθιστά τις παραδοσιακές διαδικασίες με μια πλήρως offline, σε πραγματικό χρόνο διαδικασία κινητής τεχνητής νοημοσύνης. Χρησιμοποιώντας ένα κβαντισμένο μοντέλο EfficientNet-B0 μετατραμμένο σε TensorFlow Lite, η εφαρμογή επεξεργάζεται καρέ ζωντανής κάμερας απευθείας στη συσκευή, εξαλείφοντας την εξάρτηση από cloud, την καθυστέρηση και τις ανησυχίες απορρήτου. Το σύστημα είναι βελτιστοποιημένο για συσκευές Android μεσαίας κατηγορίας, περιορίζοντας την ανάλυση σε ~2 FPS για να διατηρεί τη θερμική σταθερότητα ενώ παρέχει χρόνους πρόβλεψης <600ms.
